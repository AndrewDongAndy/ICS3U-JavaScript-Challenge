ICS 3U Hangman

JavaScript Challenge Assignment

This project was made as my submission to the JavaScript Programming Challenge in the course ICS 3U.

The link to my repository on GitHub for this project can be found here .

How to play

  1. Select a game mode
    • regular game: a word is randomly selected as the answer
    • custom game: set an answer and let a friend try to guess it
  2. Guess letters (one character, alphabetic) which are in the answer
    • press enter to submit
    • if your guess is not in the answer, you receive an incorrect guess!
  3. Or, try to guess the whole phrase at once!
